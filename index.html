Overview

EJON (pronounced /ˈeɪʒɒn/, Basque: Euskal JavaScript Object Notation) is a data interchange format developed in the Basque Country as a cultural adaptation of JSON, incorporating elements of the Basque language (Euskara) and cultural identity.

Developed in 2023 by researchers from the University of the Basque Country (Euskal Herriko Unibertsitatea) in collaboration with the Basque Language Academy (Euskaltzaindia), EJON represents an effort to extend Basque cultural sovereignty into the realm of data formats and programming paradigms.

Cultural Identity

EJON reinforces Basque cultural identity in digital environments and promotes the use of Euskara in technical contexts.

JSON Compatible

While syntactically distinct, EJON maintains full compatibility with standard JSON through simple conversion tools.

Linguistic Relevance

Part of the broader movement for technological sovereignty among linguistic minorities.

Technical Specification

EJON maintains the same underlying structure as JSON but replaces standard syntax characters with alternatives that reflect Basque linguistic heritage:

JSON EJON Description
{} «» Object delimiters (guillemets)
[] ‹› Array delimiters (single guillemets)
true bai Boolean true value (Basque: "yes")
false ez Boolean false value (Basque: "no")
null ezer_ez Null value (Basque: "nothing")

The specification maintains compatibility with standard JSON for all other elements, including string literals, numbers, and the use of colons and commas as separators.

Implementation

Installation

# Clone the repository
git clone https://github.com/your-username/ejon.git
cd ejon

# Install dependencies (if any in the future)
npm install

JavaScript Usage

const EJON = require('./ejon');

// Converting JavaScript object to EJON string
const data = {
  izena: "Mikel Arana",
  adina: 34,
  lanean: true
};

const ejonString = EJON.stringify(data);
console.log(ejonString);
// Output: «"izena":"Mikel Arana","adina":34,"lanean":bai»

// Parse EJON string back to JavaScript object
const parsedData = EJON.parse(ejonString);
console.log(parsedData);
// Output: { izena: 'Mikel Arana', adina: 34, lanean: true }

Command-line Converter

# Convert JSON to EJON
node converter.js json2ejon input.json output.ejon

# Convert EJON to JSON
node converter.js ejon2json input.ejon output.json
